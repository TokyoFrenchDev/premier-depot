Memo Commandes GIT

-git init permet de configurer un dossier avec git -un fichier caché apparait .git est créé Aller dans file>préférences>setting>écrire file.exclude et supprimer .git si le fichier n'apparait pas. -git add "nomdufichier" pour marquer le fichier modi0fié par exemple add "index.html" -git status permet de vérifier l'état du fichier -git commit -m "nomducommit" permet de faire un instantané du fichier par exemple git commit -m "Add Title" -git log permet de voir tous les instantanés déja crées. PENSER A FAIRE REGULIEREMENT DE commit Voici les différentes ETAPES du fichier : - A la création du fichier, il est Untracked. - Apres Git add, il est Indexé. - Apres git commit, il est Validé. etc
